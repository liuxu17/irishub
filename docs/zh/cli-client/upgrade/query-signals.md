# iriscli upgrade query-signals

## 描述

查询软件升级过程中signal的信息

## 用法

```
iriscli upgrade query-signals
```

打印帮助信息:

```
iriscli upgrade query-signals --help
```

## 例子

查询软件升级过程中signal的信息

```
iriscli upgrade query-signals
```

```
[
  "fca1pz46rw68wjxklxp7ptw6wrdvn5hae04jhuhdc5",
  "fca1uvjn440xjgech86avm855v5u5c65dar6gu3ng4"
]
```
