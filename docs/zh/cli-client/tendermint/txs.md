# iriscli tendermint txs

## 介绍

搜索查询所有符合指定匹配条件的交易

## 用法

```
iriscli tendermint txs [flags]
```

## 标志

| 名称，速记 | 默认值              |功能介绍                                                             | 是否必填     |
| --------------- | -------------------- | --------------------------------------------------------- | -------- |
| --chain-id      | ""                   | 区块链网络ID   | yes     |
| --node string   | tcp://localhost:26657| 节点查询rpc接口|
| --help, -h      |                      | 帮助信息 |    |
| --trust-node    | true                 | 是否信任查询节点     |          |
| --tags          | ""                   | 匹配条件     |          |
| --page          | 0                    | 分页的页码     |          |
| --size          | 100                  | 分页的大小     |          |

## 示例

### 查询交易

```shell
iriscli tendermint txs --tags `action=send&sender=faa1c6al0vufl8efggzsvw34hszua9pr4qqyry37jn` --chain-id=irishub-test --trust-node=true
```
