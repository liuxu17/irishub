# iriscli service refund-fees 

## 描述

从服务费退款中退还所有费用

## 用法

```
iriscli service refund-fees [flags]
```


## 示例

### 从服务费退款中退还费用 
```shell
iriscli service refund-fees --chain-id=test --from=node0 --fee=0.004iris
```
