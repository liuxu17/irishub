# iriscli service withdraw-fees 

## 描述

从服务费收入中取回所有费用

## 用法

```
iriscli service withdraw-fees [flags]
```


## 示例

### 从服务费收入中取回费用 
```shell
iriscli service withdraw-fees --chain-id=test --from=node0 --fee=0.004iris
```
